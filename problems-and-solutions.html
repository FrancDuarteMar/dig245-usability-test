Problems and Solutions

1. The integrated search bar

The problem

This search bar is quite restrictive as it only displays certain options when the user searches a term up. The searched term provides a list of possible topics that can be expanded to see more but the user is restricted to only these topics. Next, the user can only see a small list of possible articles. There is no way to expand the list of stories.

The solution

The current search bar is great for someone who knows what theyâ€™re looking for but for those with a broader search query, it would be best to dedicate a page that will list all articles instead of only a few. It gives the user a larger selection to explore.

2. Join the mailing list

The problem

The mailing list is an annoying pop up that distracts the user from using the site. It popped up various times while the user was attempting to click on an article. Additionally, the mailing list was not able to be accessed by my computer, only the test subject's computer.

The solution

In order to make the mailing pop up less intrusive, make the decline an ask me later and create a new option that will stop showing the mailing list pop up forever. Additionally, add an option to sign up to the mailing list in the footer. It will allow the user to sign up at any time even if they opt out of the mailing list pop up

3. Use the accessibility features to navigate to another article

The problem

The user was not able to navigate the page using a keyboard because the accessibility pop up was blocking the user from continuing until they pressed the Tab key multiple times. This pop up is good for those that want to use other accessibility features because it provides more in depth features but it confuses users due to its small size.

The solution

Since the user did not notice the accessibility features pop up, one solution to this problem can be to enlarge the accessibility pop up into one that looks similar to the mailing list pop up. This can also allow the user to select accessibility features that they want to enable on this website. The only time this would pop up would be with the screen readers, caret browsing key or multiple tab key presses in a repeated sequence.
